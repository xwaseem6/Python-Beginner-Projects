# Login System 🔐

A simple command-line login system built in Python.

## Features

- User login with username and password
- Username and password validation
- Maximum of 3 login attempts
- Displays a success message on valid login
- Prevents unlimited login attempts

## Concepts Practiced

- Variables
- User Input (`input()`)
- `if` / `else`
- `while` loops
- Logical Operators (`and`)
- `break`
- Functions (if used)
