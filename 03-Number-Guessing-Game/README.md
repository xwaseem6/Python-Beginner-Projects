# Number Guessing Game

A simple command-line number guessing game built in Python.

The program generates a random number between **1 and 10**, and the player has **5 attempts** to guess it correctly. After each guess, the program provides feedback to help the player find the correct number.

---

##  Features

- Generates a random number between 1 and 10
- Gives the player 5 chances to guess the number
- Displays whether the guess is too high or too low
- Validates user input (only accepts numbers between 1 and 10)
- Does not count invalid inputs as attempts
- Displays the total number of attempts when the player wins
- Reveals the correct number if the player runs out of attempts
