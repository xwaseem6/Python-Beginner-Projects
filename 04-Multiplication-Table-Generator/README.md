# Multiplication Table Generator

A simple command-line multiplication table generator built in Python.

The program takes a number and a range from the user, then displays the multiplication table for that number up to the specified range.

---

## Features

* Accepts a number from the user
* Accepts a custom range for the multiplication table
* Displays the multiplication table from **1** to the specified range
* Uses a function to generate the table
* Produces clean and readable output
